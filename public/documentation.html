myFlix

myFlix application will provide users to access the information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of All movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET :title None A JSON object holding data about the selected movie containing description, genre, director and img URL. Example: 
   
                    
                        {
                            Title: '',
                            Genre: {
                                Name:'',
                                Description: ''
                            },
                            Director: {
                                Name: '',
                                Birth: '',
                                Death: '',
                                Bio: ''
                            },
                            ImageUrl: 
                        }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/[genreName] GET :genreName None A JSON object holding data about the selected genre. Example: 
    
                    
                        {
                            name: 'Thriller', 
                            description: ''
                        }
Return data about a director (bio, birth year, death year) by name /movies/directors/[name] GET :directorName None A JSON object holding data about the selected director. Example: 
                    
                        {
                            name: 'Rajkumar Hirani',
                            bio: ''
                            birth: '1962', 
                            death: '-',
                        }

Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
                    
                        { 
                            "username": "anna",

Allow users to update their user info (username) /users/[id] PUT :id A JSON object holding the new username. Example: 
                    
                        { 
                            "username": "Mary"
                        }
A JSON object with the updated user profile. Example: 
                    
                        { 
                            id: '1',

                            password: 'stellers68@23'

                            ]

Allow users to add a movie to their list of favorites /users/[userid]/favourites/[movieid] PATCH :id, :favourites None A text message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/[userid]/favourites/[movieid] DELETE :id, :favourites None A text message indicating the movie was deleted
Allow existing users to deregister /users/[id] DELETE :id None A message indicating the user was deregistered
